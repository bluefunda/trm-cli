# trm-cli
trm cli

Run following CLI commands:
1.	Login : # trm login ( enter name username and password ) #
   
3.	Health : trm health
   
5.	Set SAP server URL: trm set url <sap-url>

6.	Read SAP users: trm sap-user read all or trm sap-user read <username>

7.	Create SAP user: trm sap-user create <username>

8.	Clone SAP user : trm sap-user clone <usernameCloneFrom> <NewUsername>

9.	Unit test: trm unit-test <package>

10.	Code inspector: trm qa-check <”package” or “object”> <value>

11.	Read git repository: trm repo read
    
13.	Set git repository: trm repo set <repoName>

14.	Git add: trm git add <”.” Or “objectname”>
    
16.	Git push : trm git push
